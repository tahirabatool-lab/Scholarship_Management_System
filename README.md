# 🎓 Scholarship Management System (ScholarPK)

A complete web-based Scholarship Management System developed using **PHP and MySQL**.  
This system helps manage scholarship listings, student applications, document uploads, and admin approvals in a structured and secure way.

---

## 🌐 Live Project

- 🚀 Live Website: https://smsportalpk.infinityfreeapp.com/  
- 📂 GitHub Repository: https://github.com/tahirabatool-lab/Scholarship_Management_System.git  

---

## 🎯 Project Purpose

The purpose of this project is to digitalize the scholarship application process by providing a centralized system where:

- Students can apply for scholarships online  
- Admin can manage scholarships and applications  
- Documents and notifications are handled digitally  
- Application status can be tracked easily  

---

## 👥 User Roles

### 🧑‍🎓 Student Panel
- Register and login system  
- Browse scholarships  
- Apply for scholarships  
- Upload required documents  
- Track application status  
- View notifications  
- Manage profile  

### 🧑‍💼 Admin Panel
- Admin dashboard with statistics  
- Add, edit, delete scholarships  
- Review and manage applications  
- Approve or reject applications  
- Manage users (students)  
- View contact messages  
- Send notifications  

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript, Bootstrap  
- Backend: PHP (Core PHP)  
- Database: MySQL  
- Server: Apache (XAMPP / InfinityFree)  

---

## 📦 Main Features

- User authentication system (login/register/logout)  
- Role-based access (Admin & Student)  
- Scholarship CRUD system  
- Application submission system  
- Document upload system  
- Notification system  
- Contact form system  
- Admin reporting dashboard  
- Application status tracking  

---

## 🗄️ Database Tables

- users  
- scholarships  
- applications  
- application_documents  
- notifications  
- contact_messages  
- payments  
- activity_logs  

---

## 🔐 Security Features

- Password hashing using bcrypt  
- Prepared statements (SQL Injection protection)  
- Session-based authentication  
- Role-based access control  
- File upload validation  
- Input sanitization
